# 🍄 Slope Flip Quest

This is a simple interactive app I created for **CS351 Assignment 1, Question 3** at Minerva University.

The goal of this app is to make one statistical idea easier to see:  
even if the original regression line has a positive slope, adding one extreme point can change the direction of the line and flip the slope to negative.

Instead of only reading about this idea, I wanted to make it more visual and intuitive.

## What the app does

- lets the user choose a positive slope and intercept
- creates a scatterplot with 4 points
- allows the user to explore where a 5th point could be placed
- shows how one extreme point can flip the regression line

## What this app helps illustrate

Through this app, I wanted to better understand:

- how OLS regression responds to extreme points
- why outliers can have strong influence on a fitted line
- how one additional observation can change the overall pattern in the data

## How to use it

No installation is needed.  
Just open `slope_flip_game.html` in a browser.
